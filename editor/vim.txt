Modes in Vim :-

Vim has different modes â€” understanding them is the key to using Vim effectively:

  +------------------+--------------------------------------------+---------------------------------------------------+
  | Mode             | Description                                | How to Enter                                      |
  +------------------+--------------------------------------------+---------------------------------------------------+
  | Normal Mode      | Default mode for navigation and commands   | Press `Esc`                                       |
  | Insert Mode      | Used for typing text                       | Press `i`, `a`, or `o`                            |
  | Visual Mode      | Used for selecting text                    | Press `v` (char), `V` (line), or `Ctrl+v` (block) |
  | Command Mode     | Used for saving, quitting, searching, etc. | Press `:` from Normal mode                        |
  +------------------+--------------------------------------------+---------------------------------------------------+

Starting Vim :-  

commands :- vim filename
If the file exists, it opens it.
If not, it creates a new file with that name.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Example Vim Workflow :- 

  1) vim hello.c        # open a file
  2) i                  # enter insert mode
  3) (type your code)
  4) Esc                # return to normal mode
  5) :w                 # save
  6) :q                 # quit
  7) :wq                # save + quit

1) Basic Insert Commands :-  

  +---------+--------------------------------------------+
  | Command | Action                                     |
  +---------+--------------------------------------------+
  | i       | Insert text before the cursor              |
  | a       | Insert text after the cursor               |
  | o       | Open a new line below and start inserting  |
  | O       | Open a new line above and start inserting  |
  | Esc     | Exit insert mode and return to normal mode |
  +---------+--------------------------------------------+

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------

2) Navigation (Movement) Commands :-  

  +---------+--------------------------+
  | Command | Action                   |
  +---------+--------------------------+
  | h       | Move left                |
  | j       | Move down                |
  | k       | Move up                  |
  | l       | Move right               |
  | 0       | Move to start of line    |
  | $       | Move to end of line      |
  | w       | Jump to next word        |
  | b       | Jump to previous word    |
  | gg      | Go to first line of file |
  | G       | Go to last line of file  |
  | :n      | Go to line number *n*    |
  +---------+--------------------------+

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------

3) Editing Commands :-  

  +----------+---------------------------+
  | Command  | Action                    |
  +----------+---------------------------+
  | x        | Delete a single character |
  | dd       | Delete current line       |
  | yy       | Copy (yank) current line  |
  | p        | Paste below cursor        |
  | P        | Paste above cursor        |
  | u        | Undo last change          |
  | Ctrl+r   | Redo undone change        |
  | cw       | Change current word       |
  | cc       | Change entire line        |
  +----------+---------------------------+

5) File Commands :-  

  +---------------+----------------------------------+
  | Command       | Action                           |
  +---------------+----------------------------------+
  | :w            | Save file                        |
  | :q            | Quit                             |
  | :q!           | Quit without saving              |
  | :wq or :x     | Save and quit                    |
  | :e filename   | Open another file                |
  | :r filename   | Insert contents of another file  |
  | :!command     | Run shell command (e.g., `:!ls`) |
  +---------------+----------------------------------+

6) Visual Mode Commands :-  

  +----------+-------------------------------+
  | Command  | Action                        |
  +----------+-------------------------------+
  | v        | Start visual mode (character) |

  | Ctrl+v   | Start block visual mode       |
  | y        | Copy selection                |
  | d        | Delete selection              |
  | >        | Indent selection              |
  | <        | Unindent selection            |
