SCP (Scecure Copy Protocol) :- 

  1] It is a command-line tool used for securly transfering files between a local and a remote host, or between two remote hosts,
     using SSH for Encryption and Authentication.

  Command :-      scp <localFile> [username]@[hostname or IP Address]:<hostLocation>
 
                  [username]                :-  Remote Server username.
                  [hostname or IP Address]  :-  Server's hostname of IP address.
